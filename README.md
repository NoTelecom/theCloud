## OURCLOUD

> 仿网盘项目

### 启动wangpan项目

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```

### 启动ejs项目

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run devstart
```
### 启动MongoDB服务

ejs项目显示连接成功后即可开始注册使用
